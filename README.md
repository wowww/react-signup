# React Authentication

## 구동 방법
1. firebase에서 고유한 키 값으로 .env 파일 만들기  
2. npm install
3. npm start

--- 

## 기능 
1. 이메일 암호 인증(등록/로그인)  
2. Google 로그인 기능  
3. 비밀번호 찾기 기능  
4. 사용자 정의 암호 재설정 페이지  
5. Chakra UI를 활용한 다크 모드 템플릿
